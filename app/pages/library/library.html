Library

This is a link to a library of effects and sound generators made by the SunVox community. All are sorted by type and then by user. It includes everything shipped with the latest distribution of SunVox. The files themselves are saved in sunsynth format (for a description of sunsynth files see How to Save and Load Files).

The collections is saved to github here, you can navigate the directory, download individual files, or download the entire collection at once (see the green "clone or download" button.) If you are familiar with github, you can clone it and pull to get updates.
